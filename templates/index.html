Technologies Used

Tensorflow

I have used Tensorflow 1.12 for model training.
Tensorflow model server REST API has been used for real time prediction.

Apache Spark

I have used Spark 2.3.2 and Spark streaming for real time data processing operation.
Two type of metrics are getting calculated here. 1. Per minute twitter count 2. Per minute positive/negative sentiment.

Apache Kafka

A single broker Kafka cluster has been used to consume Twitter data from Twitter API.
And topic messages are being fetched by Spark Streaming job.
